# 登录与权限说明

进入首页前需先登录，所有数据接口（含数据底表、数据维护等）均需有效登录态方可访问。

---

## 默认管理员账号

已为你创建**管理员账号**（首次部署时由 `deploy/create_admin.py` 生成）：

| 项目 | 值 |
|------|-----|
| **用户名** | `admin` |
| **默认密码** | `Admin123!` |

**建议**：首次登录后若需修改密码，在项目根目录执行：

```bash
python deploy/create_admin.py admin 新密码
# 或交互式输入：
python deploy/create_admin.py
```

会覆盖 `deploy/auth_users.json`，仅保留本次创建的用户（可多次执行以重置密码）。

---

## 行为说明

- **未登录**：打开 `/frontend/` 会先显示登录框，输入正确用户名和密码后进入系统。
- **已登录**：Session 通过 Cookie（`slg_session`）保存，有效期 7 天；关闭浏览器后再次打开若未过期可直接进入。
- **退出**：点击右上角「退出」会清除登录态并刷新页面，需重新登录。

---

## 技术说明

- 用户信息存放在 `deploy/auth_users.json`（用户名 + 盐 + 密码哈希）。
- 密码使用 SHA256(盐 + 密码) 存储，不存明文。
- 除 `GET /api/auth/check` 与 `POST /api/auth/login` 外，其余 `/api/*` 请求均会校验 Session，未登录返回 401。
